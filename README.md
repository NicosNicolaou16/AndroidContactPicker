This open-source project demonstrates the implementation of the new **Contact Picker** in Android using Jetpack Compose. Introduced as a privacy-first feature in **[Android 17](https://developer.android.com/about/versions/17/features/contact-picker)**, this API allows users to select specific contacts or contact details to share with an app without granting the app full access to their entire contact list.

## 🚀 Getting Started

To run this project, you must set up the **Android CinnamonBun (Android 17 Preview)** environment:

1.  **Install Android Studio:** Use the latest **Canary** or **Preview** version.
2.  **Download SDK Platform:**
    *   Go to `Settings > Languages & Frameworks > Android SDK`.
    *   Install **Android SDK Platform CinnamonBun** (API Level: 35+ / CinnamonBun).
3.  **Install Build Tools:**
    *   In the **SDK Tools** tab, ensure **Android SDK Build-Tools 37.0.0-rc2** (or newer) is installed.
4.  If you don’t have a physical Pixel device, test the API using the Android Emulator by creating a Virtual Device with the **64-bit CinnamonBun (Android 17) system image**.

## 🛠️ Tech Stack & Libraries

This project is built with **[Kotlin](https://kotlinlang.org/docs/getting-started.html)** and utilizes modern Android libraries:

-   **UI:** [Jetpack Compose](https://developer.android.com/develop/ui/compose)
-   **API:** [Contact Picker API](https://developer.android.com/about/versions/17/features/contact-picker) (Privacy-focused system picker)

## 🔧 Versioning

-   **Target SDK:** **37** (Android 17 requirement)
-   **Minimum SDK:** **28**
-   **Kotlin Version:** **2.3.20**
-   **Gradle Version:** **9.1.0**
